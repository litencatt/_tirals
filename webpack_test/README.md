## yarn
```
$ brew update
$ brew install yarn
```
## Webpack

```
$ yarn init -y
$ yarn add webpack webpack-dev-server --dev

(このディレクトリにあるファイルを用意)

$ yarn build
$ yarn start

localhost:8080にアクセスするとalertが実行される

### すぐ動作確認したい人
このレポジトリをpullしたあとに
$ yarn
$ yarn build
$ yarn start
localhost:8080にアクセス
```
